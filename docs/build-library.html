To run pyrlang you need to have term installed. Term is an external package that pyrlang uses to encode / decode Erlangs external term fomrat. Install it with:

pip3 install pyrlang-term

To install Pyrlang you checkout the project and install it:

git clone git@github.com:Pyrlang/Pyrlang.git
cd Pyrlang
pip install .

Install requirements via: pip3 install -r requirements.txt, this will also pull requirements for Sphinx documentation generator (not needed if you only use the library).

To build the docs you need to install Sphinx and some other pacakges you can do that with the requirements file:

pip install -r requirements.txt

Source for the documentation is in the docs-src/ directory. To generate HTML, run make docs and the docs directory will be deleted and recreated.
